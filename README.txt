SpaceX Launch API Project! This will eventually get more features but am currently working on another project that has more priority at the moment<br/><br/>
<strong>What this does currently</strong><br/>
- Retrieves SpaceX Launches (v5)<br/>
- Displays Launch data in a chart with filters<br/>
- "Load more" on demand and loads 10 launches per load<br/>
<br/>

- Expanding chart filters<br/>
- Filter / sorting Launches<br/>
- Component for upcoming launches in a quick overview<br/>
- More relevant information in the Launch cards<br/>
- Dedicated launch details view<br/>
